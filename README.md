# Code-for-network-analysis-of-anxiety-and-depression-symptoms-among-adults-with-chronic-pain
# Anxiety–Depression Symptom Network in Adults with Chronic Pain

This repository contains the analysis code used in the study:

**“Anxiety–depression symptom network among U.S. adults with chronic pain: A population-based network analysis.”**

## Data source
Data were obtained from the **2019 National Health Interview Survey (NHIS)** conducted by the National Center for Health Statistics (NCHS).

NHIS website:
https://www.cdc.gov/nchs/nhis

The NHIS dataset is publicly available. Raw data are not included in this repository and should be downloaded directly from the official website.

## Software
All statistical analyses were performed using:

- **Python (version 3.9.7)**
- **R (version 4.4.1)**

### Python analysis
Python was used for data preprocessing, descriptive statistics, network estimation, and visualization.

Main Python libraries:
- pandas
- numpy
- scipy
- NetworkX
- scikit-learn
- matplotlib
- seaborn

### R analysis
R was used for gender-specific subgroup analyses and the Network Comparison Test (NCT).

Main R packages:
- bootnet
- qgraph
- reshape
- NetworkComparisonTest
